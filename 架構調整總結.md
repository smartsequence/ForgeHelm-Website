## 💡 關鍵決策回顧

### Q1: 官網應該建在 Azure 還是 Vercel？

**答案：Vercel** ⭐

**原因：**
1. 官網是靜態網站，Vercel 更適合
2. 極簡部署，一鍵完成
3. 速度更快（Vercel Edge Network）
4. 完全免費（100 GB 流量足夠）
5. 不影響 Azure 上的 SaaS 服務

**架構分離：**
```
官網（展示）      → Vercel        → smartsequence.tech
SaaS（應用）      → Azure         → app.docengine.com
API（SignalR）    → Azure         → api.docengine.com
```

### Q2: 單一產品官網 vs 工作室官網？

**答案：工作室官網** ⭐

**原因：**
1. 更具擴展性（未來可能有其他產品）
2. 建立品牌形象（智序資訊工作室）
3. 展示專業度（創辦人背景、技術實力）
4. 更容易建立信任（統編、公司資訊）
5. SEO 更好（更多內容層級）

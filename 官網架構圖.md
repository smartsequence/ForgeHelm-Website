## 📊 資料流程圖

### 使用者訪問流程

```
使用者輸入網址
    │
    ▼
smartsequence.tech
    │
    ├─→ DNS 解析
    │       │
    │       ▼
    │   Vercel Edge Network
    │       │
    │       ▼
    │   最近的 CDN 節點
    │       │
    │       ▼
    │   靜態 HTML/CSS/JS
    │       │
    │       ▼
    └─→ 瀏覽器渲染
            │
            ├─→ 瀏覽產品資訊
            ├─→ 查看定價方案
            ├─→ 填寫諮詢表單 → Web3Forms → Email 通知
            └─→ 點擊「免費試用」→ app.docengine.com
```

### 表單提交流程

```
使用者填寫表單
    │
    ▼
前端驗證（JavaScript）
    │
    ▼
提交到 Web3Forms API
    │
    ├─→ 成功
    │   │
    │   ├─→ 發送 Email 到 contact@smartsequence.tech
    │   ├─→ 發送確認信給使用者
    │   └─→ 顯示成功訊息
    │
    └─→ 失敗
        │
        └─→ 顯示錯誤訊息
```

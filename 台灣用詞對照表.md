## 📌 撰寫原則

### 優先使用台灣用詞
- ✅ AI 智慧文件生成
- ✅ 多管道聯絡方式
- ✅ 線上服務
- ✅ 使用者體驗
- ✅ 資料庫分析

### 避免使用大陸用詞
- ❌ AI 智能文檔生成
- ❌ 多渠道聯繫方式
- ❌ 在線服務
- ❌ 用戶體驗
- ❌ 數據庫分析

### 例外情況
某些技術術語兩岸通用，無需特別調整：
- Git、GitHub、Repository
- API、REST、GraphQL
- Docker、Kubernetes
- CI/CD、DevOps
- AWS、Azure、GCP

## 🔍 檢查指令

使用以下 PowerShell 指令檢查是否有遺漏的大陸用詞：

```powershell
# 檢查「智能」
Select-String -Path "*.md","*.ts","*.astro" -Pattern "智能" -Recurse

# 檢查「渠道」
Select-String -Path "*.md","*.ts","*.astro" -Pattern "渠道" -Recurse

# 檢查「信息」
Select-String -Path "*.md","*.ts","*.astro" -Pattern "信息" -Recurse

# 檢查「數據」（需排除"資料庫"）
Select-String -Path "*.md","*.ts","*.astro" -Pattern "數據" -Recurse

# 檢查「用戶」
Select-String -Path "*.md","*.ts","*.astro" -Pattern "用戶" -Recurse

# 檢查「文檔」
Select-String -Path "*.md","*.ts","*.astro" -Pattern "文檔" -Recurse
```

## ✏️ 特別注意

### 容易混淆的詞
1. **「數據」vs「資料」**
   - 大部分情況用「資料」
   - 資料分析 ✅
   - 數據分析 ❌

2. **「文檔」vs「文件」**
   - 技術文件 ✅
   - 技術文檔 ❌
   - API 文件 ✅
   - API 文檔 ❌

3. **「智能」vs「智慧」**
   - AI 智慧 ✅
   - AI 智能 ❌
   - 智慧型手機 ✅
   - 智能手機 ❌

4. **「賬號」vs「帳號」**
   - 使用者帳號 ✅
   - 用戶賬號 ❌
   - 登入帳號 ✅
   - 登錄賬號 ❌
